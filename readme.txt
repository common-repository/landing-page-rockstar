Create beautiful, high converting landing pages or squeeze pages in seconds. Plugin includes pre-built templates and opt-in code functionality.

== Description ==

** Update 0.5 Important Note ** This is a major update that changes much in the way that landing pages are rendered including web forms. After upgrading, please double check all of your landing pages to make sure they are still functioning.

**Please Note:** This is a beta testing version of this plugin. Therefore there may be some bugs that you come across. Please report any bugs through my [contact form here](http://www.ericsestimate.com/contact-eric/ "Contact Form").

**Your feedback will help me to improve the plugin in future releases!**

== Installation ==

1. Upload the entire lprockstar directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. A new post type will appear in your menu on the left side of your screen. From here, you can create a new landing page.
